# graphiql-desktop

> Electron-based desktop application for GraphiQL 2

![GraphiQL Desktop](./assets/screenshot.png)

## Shortcuts

`⌘D` - Show `Docs`

`⌘K` - Search

`⌘Y` - Show `History` View

`⌘E` - Show `Explorer` View

`⌘R` - Reload Schema

`⌘T` - New Tab

`⌘W` - Close Tab

`⌘1` through `⌘9` - Show Tab `N`

`⌘0` - Show Last Tab
